**Категория A: Консольные утилиты и простые приложения (Недели 2-7)**
1.  **Калькулятор с историей:** Консольная программа, которая запоминает последние 5-10 операций и позволяет их просматривать.
2.  **Конвертер валют/единиц измерения:** Использует фиксированные курсы/коэффициенты или простой API (на подобие `requests`). Ввод: `100 USD to EUR`.
3.  **Викторина:** Программа задает вопросы из файла (JSON/CSV), ведет счет, выводит результат в конце.
4.  **Игра "Угадай число":** Компьютер загадывает число, игрок угадывает. Реализовать уровни сложности (диапазон чисел), подсчет попыток.
5.  **Текстовый квест:** Простая игра с ветвлениями, где выбор игрока влияет на развитие сюжета. Данные можно хранить в словарях.
6.  **Генератор безопасных паролей:** Пользователь задает длину, использование букв, цифр, спецсимволов. Пароль генерируется случайным образом.
7.  **To-Do List (Консольная версия):** Добавление, удаление, отметка выполненных задач, сохранение списка в файл и загрузка из него.
8.  **Анализатор текста:** Загружает текст из файла, выдает статистику: количество слов, предложений, уникальных слов, самое частое слово.
9.  **Адресная книга/Телефонный справочник:** Хранение контактов (имя, телефон, email) в словаре или списке словарей. Поиск, добавление, удаление, экспорт в файл.
10. **Игра "Камень, ножницы, бумага" с ИИ:** Простой ИИ, который "запоминает" ходы игрока и пытается предугадать следующий. Ведение статистики.
